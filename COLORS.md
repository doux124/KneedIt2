# Color Palette

| Role            | Hex       | Swatch |
|-----------------|-----------|--------|
| Primary         | `#D1E2F1` | 🟦 light blue |
| Text            | `#394955` | ⬛ slate gray |
| Secondary       | `#8AB0CD` | 🟦 muted blue |
| Accent          | `#F5D8E7` | 🟪 soft pink |
| Accent (alt)    | `#BDA1B0` | 🟪 mauve |

## Notes

- `#D1E2F1` is the **primary** color.
- `#394955` is the **text** color.
- `#8AB0CD` is the **secondary** color.
- `#F5D8E7` is an **accent** color.
- `#BDA1B0` is an **accent** color.

> **Corrections:** Two values were written with the letter `O` instead of the
> digit `0` (hex codes only allow `0`–`9` and `A`–`F`):
> - `#8ABOCD` → `#8AB0CD`
> - `#BDA1BO` → `#BDA1B0`
